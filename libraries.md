# Required Libraries

Install these libraries using the Arduino IDE Library Manager.

## External Libraries

- Adafruit GFX Library
- Adafruit SSD1306
- Keypad
- ArduinoJson
- BleKeyboard

## Built-in ESP32 Libraries

These come with the ESP32 Board Package.

- WiFi
- HTTPClient
- Wire
- WiFiClientSecure
- WebServer
- time

## Board Package

ESP32 by Espressif Systems
