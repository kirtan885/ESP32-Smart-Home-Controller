# Wiring Guide

## Controller ESP32

OLED

| OLED | ESP32  |
| ---- | ------ |
| VCC  | 3.3V   |
| GND  | GND    |
| SDA  | GPIO21 |
| SCL  | GPIO22 |

Keypad

| Row | GPIO |
| --- | ---- |
| R1  | 13   |
| R2  | 4    |
| R3  | 14   |
| R4  | 27   |

| Column | GPIO |
| ------ | ---- |
| C1     | 26   |
| C2     | 25   |
| C3     | 33   |
| C4     | 32   |
